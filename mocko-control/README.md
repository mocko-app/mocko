# @mocko/control

Control Panel for [Mocko](https://github.com/mocko-app/mocko).

This package contains the Next.js UI and API routes used to manage mocks, hosts, flags, and management operations.

Most users should install `@mocko/cli` instead:

```sh
npm i -g @mocko/cli
mocko
```

## Used By

- `@mocko/cli`
- Mocko Docker images
